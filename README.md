# What's Aggr?
The server engine developed using C#, support Http, Socket (TCP/UDP) protocol, provide Redis, SQL, MySql, SQLite database.

# Requirements

.Net Framework 4.0, 4.5.


# Links

Official website: [http://www.aggrengine.com/](http://www.aggrengine.com/)


# Features

## Network

* Http

* Socket(TCP/UDP)

## Storage

* RDS
    * SQL
    * MySql
    * SQLite for unity3d

* NoSql
    * Redis

* File
    * Excel
